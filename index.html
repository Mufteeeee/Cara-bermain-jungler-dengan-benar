Cara Bermain Jungler Dengan Baik

=Langkah2 Nya Adalah=

Ada 2 cara farming di fase2 awal game

Cara 'Aman' Dan 'Resiko'

AMAN

    1.Farming ke blue buff
    2.Farming Ke red buff

RESIKO

-Sama hal nya dengan gambar di atas tapi bedanya di wilayah musuh

-Jika ingin mengambil nya kalian harus penuh dengan perhitungan dan ke waspadaan

-Karena jika kalian ketahuan dan dibunuh itu akan menguntungkan musuh

-Jika ingin merusuh atau merebut buff musuh sebaiknya kalian membawa rekan tim kalian walaupun resikonya tetap besar

Langkah Kedua

-Alangkah baiknya jungler harus menggangbang marskman musuh agar mm musuh tidak mendapatkan pendapatan untuk lategame

-Jika mm musuh di atas start buff merah, jika mm musuh di bawah start buff biru.

-Dapatkan level4 secepat mungkin agar tidak tertinggal jungler musuh

Langkah Ketiga

-Set up turle

-Di usahakan setelah level 4 kalian memanggil rekan tim untuk set up turtle

-EXP lane kalian harus ngezoning damage dealer belakang lawan atau jungler musuh atau di usahain di pick off

-ROAMER kalian harus bisa nge set up atau ngebuka map agar kita sebagai jungler biasa leluasa hit turtle

-Dan kita sebagai jungler usahain retribution on point agar tim mendapatkan kebutuhan agar asumsi late game lebih besar

Langkah Ketiga

-Di usahain mulai2 menit 5 ke 8 harus snowball karena lord spawn di menit ke 8

-Sama hal nya dengan turtle tugas masing2 lane masih sama tapi sesuai dengan situasi juga

-Jika menggunakan jungler assasin atau jungler damage di usahain jika ingin set up lord jangan sampai terlihat di map

-Jika menggunakan jungler tebal kalian bisa tbin tim kalian dan bisa maju paling depan karena selain bisa nahan badan dia bisa buat nge retri.

Langkah Terakhir

-Jika sudah snowball jangan sampai blunder karena akan merugikan tim.

-Jika sudah mendapatkan lord di usahakan push barengan di 3 lane agar agar musuh dpt tekanan.

-Kita sebagai jungler hanya untuk meng execute musuh yg udah disekaratin oleh tim kita.

-Jika sudah mengikuti instruksi di atas kesempatan menang kalian akan besar.

-Inti dari semuanya adalah jangan banyak mati dan perbanyak farming
